Namelist I/O Module

Module that holds all the I/O routines to read from a setup file and setup the models and parameters included included in the thermodynamic routines

```fortran

! Namelist based input file ! ========================= ! ! Units: ! - Pressure: bar ! - Temperature: K ! - Volume: L ! =========================

&nml_setup nc=3, ! Number of components model=”PR78”, ! SRK PR76 PR78 RKPR mixrule=”ClassicVdW” ! ClassicVdW /

&nml_composition names=”C4” “C20” “H2O” spec=”critical” ! critical or parameters z=0.16 0.04 0.8 /

&nml_classicvdw kij(1, :)=0 0 0.5 kij(2, :)=0 0 0.5 kij(3, :)=0.5 0.5 0 /

&nml_critical ! Critical constants tc=425.2 782.0 647.3 pc=38.0 14.6 220.89 w=0.1928 0.816 0.344 /
